# CodeAlpha Calculator App

A modern cyberpunk-themed calculator built with HTML, CSS, and vanilla JavaScript.

## Live Links

## Overview

This project is a responsive calculator app created for the CodeAlpha Frontend Development Internship. It performs basic arithmetic operations with a clean neon interface, live result preview, button input, and keyboard support.

The project is built with beginner-friendly vanilla JavaScript and focuses on DOM handling, expression building, input validation, and responsive UI design.

## Features

- Addition, subtraction, multiplication, and division
- Real-time expression display
- Live result preview
- Clear button
- Delete button
- Decimal support
- Operator replacement logic
- Division by zero protection
- Keyboard support
- Responsive design
- Dark cyberpunk UI
- Hover, active, and focus states
- GitHub Pages deployment ready

Architecture Notes

The project is divided into three files:

index.html handles the calculator structure.
styles.css handles the cyberpunk UI, responsive layout, and button states.
script.js handles input, expression updates, live result preview, keyboard support, and final calculation.

The calculator stores the current expression in a JavaScript string and updates the display whenever the user presses a button or keyboard key.
