# CodeAlpha Calculator App

A modern cyberpunk-themed calculator built using pure HTML, CSS, and Vanilla JavaScript.

This project was developed as part of the CodeAlpha Frontend Development Internship and also satisfies the requirements for DecodeLabs Calculator Project.

---

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Real-time expression display
- Live result display
- Clear button
- Decimal support
- Operator replacement logic
- Division by zero protection
- Keyboard support
- Responsive design
- Dark cyberpunk UI
- Hover and click animations

---

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
